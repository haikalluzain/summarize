### Development

#### Requirements

- Install Git
- Install MongoDB
- Install NodeJS 20.x >=
- Install Yarn

### Install

    yarn install

### Get Started

    yarn dev

The server will run on `http://localhost:3000`

### Seeder

Go to postman and hit this endpoint to run seeder data

```http request
POST http://localhost:3000/api/seed
```
